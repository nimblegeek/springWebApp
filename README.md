# springWebApp
A practice-project to learn more about Spring Framework, version 5 and Spring Boot 2.

# Technologies used in this project

- Spring Framework 5
- Spring Boot 2
- Spring Data JPA
- Spring MVC
- Spring MockMVC
- Spring WebFlux
- Spring Web Client
- Spring Data MongoDB
- Spring RestDocs
- Hibernate
- Project Lombok
- MapStruct
- Maven
- Gradle
- JUnit 4 and JUnit 5
- Mockito
